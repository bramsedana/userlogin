# Userlogin

This repository is used for CRUD User and login

## Project setup

1. Clone this repo to go/src/github.com/bramsedana

2. Install dependencies

```
npm install
```

3. Start project

```
node run app.js
```

## User example

This project uses an online database with two users already created for testing

1.  Username: admin
    Password: adminpwd
    Role: admin

2.  Username: user
